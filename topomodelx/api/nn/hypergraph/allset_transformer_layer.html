Vertex to edge:

\[\begin{split}\begin{align*} &üüß \quad m_{\rightarrow z}^{(\rightarrow 1)} = AGG_{y \in \\mathcal{B}(z)} (h_y^{t, (0)}, h_z^{t,(1)}) \quad \text{with attention}\\ &üü¶ \quad h_z^{t+1,(1)} = \text{LN}(m_{\rightarrow z}^{(\rightarrow 1)} + \text{MLP}(m_{\rightarrow z}^{(\rightarrow 1)} )) \end{align*}\end{split}\]

Edge to vertex:

\[\begin{split}\begin{align*} &üüß \quad m_{\rightarrow x}^{(\rightarrow 0)} = AGG_{z \in \mathcal{C}(x)} (h_z^{t+1,(1)}, h_x^{t,(0)}) \quad \text{with attention}\\ &üü¶ \quad h_x^{t+1,(0)} = \text{LN}(m_{\rightarrow x}^{(\rightarrow 0)} + \text{MLP}(m_{\rightarrow x}^{(\rightarrow 0)} )) \end{align*}\end{split}\]
